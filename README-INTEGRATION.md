# UniStay Housing + Booking Integration Patch

Copy these files into your existing `unistay-frontend` branch.

Included:
- backend-ready API client
- housing API
- booking API
- professional Housing page
- professional Student booking/payment page
- professional Host booking confirmation/payment verification page

Set `.env`:

```env
VITE_API_BASE_URL=https://cdn-unistay.onrender.com/api
```

Local backend:

```env
VITE_API_BASE_URL=http://localhost:5000/api
```

Install:

```powershell
npm install axios lucide-react react-hot-toast
```
