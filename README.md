# UniStay+ Frontend

Professional React + TypeScript + Tailwind frontend using dummy data now and backend-ready API modules for later connection.

## Run

```powershell
npm config set registry https://registry.npmjs.org/
npm install --legacy-peer-deps
copy .env.example .env
npm run dev
```

Open: http://localhost:5173
